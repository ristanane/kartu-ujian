Smart Card Generator (SDN Senden 2 Edition)

Aplikasi automasi cetak kartu ujian dinamis, estetik, dan personal.

💡 Panduan Cepat Aplikasi
  • Excel: Blok kolom Nama, NISN, Tempat Lahir, Tgl Lahir, Ruang, No Peserta di Excel lalu Paste ke Langkah 1. Baris pertama wajib judul kolom.
  • Foto Otomatis: Masukkan file foto siswa ke folder foto-siswa/ di GitHub. Ubah nama file fotonya menjadi nomor NISN anak (Contoh: 012345.jpg).
  • TTD & Logo: Masukkan file gambar TTD/Stempel dan Logo ke folder aset-sekolah/, lalu panggil nama filenya di kotak input yang disediakan.
